# Sentra - Restaurant Intelligence Platform

A modern, Benchling-inspired website for Sentra, the guest demand signaling system for bars and restaurants.

## Features

- **Dark, Professional Design** - Inspired by Benchling's clean aesthetic
- **Responsive Layout** - Works perfectly on all devices
- **Pilot Signup Form** - Integrated with Formspree for lead capture
- **FAQ Section** - Addresses common questions about the platform
- **Fast & Modern** - Built with React + TypeScript + Vite

## Quick Start

```bash
npm install
npm run dev
```

## Deployment

This site is configured for easy deployment to Vercel:

1. Push to GitHub
2. Connect to Vercel  
3. Deploy automatically

## Form Setup

To set up the pilot signup form:
1. Create a Formspree account at https://formspree.io
2. Replace the form endpoint in `src/app/components/pilot-cta-section.tsx`
3. Update the email address in the form configuration
